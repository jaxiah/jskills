---
name: grill-me
description: Interview the user relentlessly about a plan or design until reaching shared understanding, resolving each branch of the decision tree. Use before writing a PRD to stress-test an initial idea, when the user wants to think through a design, or when they mention "grill me". Not a substitute for write-a-prd — use this when the idea is still rough and not yet ready to be written up.
---

Interview me relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one. For each question, provide your recommended answer.

If a question can be answered by exploring the codebase, explore the codebase instead.
